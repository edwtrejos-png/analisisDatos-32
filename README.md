# Comandos para git
## 29/06/2026
## Sesion 4

-- 1.Jalar la rama origen
git pull origin main

-- 2.Obtener el estado actual
git status

-- 3.Agregar Cambios (seleccionar los archivos / todo lo qu se debe subir)
git add .

-- 4.Crear commit
git commit -m "descrición"

-- 5.Empujar la cambios a la rama main al repositorio remoto
git push origin main
